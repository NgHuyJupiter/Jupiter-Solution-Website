# Capabilities Page Design Specifications - Overrides

This document defines page-specific design rules for the **Jupiter Solutions Capabilities Page**.

---

## 1. Structured Taxonomy
*   **Grid layout:** Certificates grouped cleanly into a structured system grid.
*   **Visual Badges:** Neutral gray backgrounds (`bg-slate-100`) switching to deep burgundy highlights (`bg-brand-50`, `text-brand-600`) upon card hover interaction.

---

## 2. Proportion Metrics (Data Visualization)
*   **Ratio Panels:** Structured horizontal metrics depicting technical staff proportions (Software Engineering, DevOps, AI Services, Cybersecurity, PMs).
*   **Execution:** Represented using absolute container widths over subtle gray background rails.
